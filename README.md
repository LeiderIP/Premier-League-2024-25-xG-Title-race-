## Features

- Select any match via **Home / Away team dropdowns**
- Navigate between all 380 matches with **arrow buttons** or keyboard arrow keys
- Cumulative xG curves with **goal markers**, player names and minute labels
- Real scoreline and xG totals displayed above the chart
- Club colours applied automatically
- Fully **self-contained HTML** — works offline once loaded
